Important FAQ

What is the DOM(Document Object Model)?

The DOM is like a tree structure of a webpage. It represents all the HTML elements (like headings, paragraphs, buttons) as objects. We can change this by using JS.

How do you select an element from the DOM?

We can use JavaScript to pick an element from the webpage:
1.document.getElementById('elementId')
3.document.getElementsByClassName('className')
2.document.querySelector('#elementId')
4.document.querySelectorAll('.className')

What is event delegation and why is it useful?

Event delegation means putting an event listener on a parent element. The parent handles clicks for all the child elements inside it.

This is useful because it allows you to handle events for multiple child elements without having to attach individual event listeners to each one.
